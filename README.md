# Wi-Tri Lab

Terminal-first, hands-on Python lab to learn Linux cybersec and ship Wi-Tri Beacon.

Clone the repo and run:
```bash
git clone <repo-url>
cd witri-lab
python3 start.py
```

Style: short tasks, open starter.py in your editor, run checks, earn badges.
